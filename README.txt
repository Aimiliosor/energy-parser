Spartacus - Energy Data Analysis Tool
=====================================
by ReVolta srl

WHAT IS SPARTACUS?
------------------
Spartacus is a desktop application for analysing, validating, and correcting
energy consumption data. It supports CSV and Excel files and provides:

  - Automatic column detection and data transformation
  - Data quality checks and validation
  - Gap filling, duplicate removal, and missing-value correction
  - Statistical analysis and branded PDF report generation
  - Export to clean Excel files

HOW TO RUN
----------
1. Open the "Spartacus" folder.
2. Double-click "Spartacus.exe" to launch the application.
3. No installation is required - the application is fully portable.

SYSTEM REQUIREMENTS
-------------------
- Windows 10 or Windows 11
- No additional software or Python installation needed
